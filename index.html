A small letter parser

The idea is that you provide some text consting of a-i characters, for example:

abc fg  hij abaa ab ab

You then do string replacements (using a text editor) and write down the steps you do, so:

  1. a -> aba
  2. b -> bbb a bb b
  3. c -> ccacc
  4. aba -> a a c aa c a a c etcâ€¦

For the final translation into chuck code, we use this webpage, for example, we can set the characters to mean the following Chuck function calls:

a = play(100);
b = play(200);
c = play(300);
d = play(400);
e = play(500);
f = play(600);
etc..






To play back this score of function calls, you need to combine it with a very small chuck program chuckProgram.ck.



SinOsc c => dac;
c.gain(0.25);

fun void play(float x) {
    x => c.freq;
    0 => c.phase;
    second / x => now;
}




This will play a sinewave segment for 1 cycle of the waveform.


You can also play around with random values:


play(Math.random2(100,1000));


You can run it by pasting it in a miniaudicle window, and clicking the green plus (make sure you start the virtual machine before you do this.
